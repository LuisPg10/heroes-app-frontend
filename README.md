# Heroes App

## Start development

1. Clone the repository.

2. Create and edit environment variables in `.env` with the environment variables in `.env.template`.

3. Execute `npm install` or `bun install` or depending of your execution environment (node, bun or deno). Remenber delete .lock file if you change of execution environment.

4. Execute `npm run dev` or with the package manager that you use .
